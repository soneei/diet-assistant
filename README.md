# 饮食分析智能体

> 不是让你忌口，是让你在能吃的范围内吃得开心。

一个基于**中医辨证 + 西医诊断交叉验证**的饮食分析与食谱定制系统。

---

## 怎么用

### 方法一：直接打开网页（最简单）🚀

👉 **[点这里打开在线版](https://soneei.github.io/diet-assistant/饮食分析智能体.html)**

1. **症状输入**：点选你的症状（怕冷/发热/胃寒/腹胀……60+细分症状）
2. **西医诊断**：填医院确诊的病名和药方
3. 点击「辨证分析 → 生成食谱」
4. 查看体质分析结果、推荐/禁忌食材、三餐建议、一周食谱、食疗方

无需安装，无需注册，浏览器打开就能用。

### 方法二：下载到本地运行

```bash
git clone https://github.com/soneei/diet-assistant.git
cd diet-assistant
open 饮食分析智能体.html   # macOS
# 或者直接双击 饮食分析智能体.html
```

### 方法三：作为 WorkBuddy 智能体使用

如果你在用 WorkBuddy，将 `SKILL.md` 导入即可在对话中使用饮食分析能力——问就能分析，吃了自动记，要就给出食谱。

## 核心功能

### 1. 问 "我能吃XXX吗？"

你问一种食物，智能体从两个角度分析：

| 维度 | 分析内容 |
|------|---------|
| 西医 | 营养成分、对当前疾病的生理影响、副作用风险 |
| 中医 | 食物性味归经（寒热温凉）、体质匹配、配伍禁忌 |
| 替代方案 | 至少2个更好吃且安全的替代食物 + 做法建议 |

### 2. 记 "刚吃了XXX，感觉……"

吃了什么告诉我，自动记录到饮食日志。长期积累形成你的**食物-身体敏感度画像**——什么食物对你是绿灯，什么是一碰就翻车的红灯。

### 3. 定 "帮我安排明天的饮食"

根据你的体质生成一日三餐 + 一周食谱，每道菜标注快乐指数。不只是"能吃什么"，更是**怎么做好吃**。

## 内置中医辨证体系

支持11种常见体质/症候的辨证分析，每种都包含完整的中西医知识库：

| 类型 | 典型症状 |
|------|---------|
| 风寒感冒 | 怕冷、发热轻、流清涕、无汗 |
| 风热感冒 | 发热、咽喉痛、流黄涕、口干 |
| 脾胃虚寒 | 胃寒、腹痛、大便稀溏、四肢冷 |
| 脾胃湿热 | 腹胀、口苦、大便黏腻、身体沉重 |
| 湿热下注 | 湿疹、皮肤痒、小便黄 |
| 食积停滞 | 腹胀、反酸、打嗝、口臭 |
| 痰湿内蕴 | 身体沉重、咳白痰、胸闷 |
| 气虚 | 乏力、气短、自汗、食欲差 |
| 阴虚 | 口干、手脚心热、失眠、夜间出汗 |
| 肝郁气滞 | 烦躁、胸闷、情绪波动大 |
| 病后恢复期 | 乏力、食欲差、脾胃最弱 |

## 竞品差异

市面上的饮食类产品（Zoe、Noom、薄荷健康、汤头APP）都没有同时做到：

- ✅ **中医辨证 + 西医诊断交叉验证**
- ✅ **有限食材内的好吃个性化**（不是只告诉你能吃什么，还告诉你怎么做才好吃）
- ✅ **对话式的持续追踪**（问就能分析，吃了自动记）
